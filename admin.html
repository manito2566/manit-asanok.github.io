📖 วิธีสร้าง Personal Access Token

1. ไปที่ github.com/settings/tokens

2. กด Generate new token (fine-grained)

3. ตั้งค่า:

  • Token name: portfolio-admin
  • Expiration: 90 days (หรือมากกว่า)
  • Repository access: เฉพาะ manit-asanok.github.io
  • Permissions → Repository → Contents: Read and write

4. กด Generate token → Copy → วางในช่องด้านบน

⚠ Token จะเก็บใน browser ของท่านเท่านั้น ไม่ส่งไปที่อื่น

📝 เขียนข่าวสาร / บทความใหม่

บทความ blog และข่าวสารใช้ WordPress เป็นระบบหลัก (มี mobile app เขียนสะดวก, comment, RSS, SEO ดี)

เปิด WordPress Editor

⏱ หลังโพสต์ใน WordPress → หน้า News ในเว็บนี้จะแสดงบทความใหม่อัตโนมัติ (ดึงผ่าน WordPress API)

📚 จัดการผลงานวิจัย

  • คลิกแท็บ "ผลงานวิจัย" ด้านบน
  • กด "เพิ่มผลงานใหม่" → กรอกข้อมูล → กด บันทึก
  • แก้/ลบได้ทันที — เว็บอัปเดตภายใน 1-2 นาทีหลังบันทึก
  • DOI ที่ใส่จะกลายเป็นลิงก์อัตโนมัติบนเว็บ

👤 จัดการ Profile

  • แก้ ตำแหน่ง / รางวัล / การฝึกอบรม ได้
  • ใส่ทั้งภาษาไทยและอังกฤษ — เว็บจะสลับตาม language switcher

🔄 Regenerate CV PDF

หลังเพิ่มผลงานหรือแก้ profile แล้ว ควร regenerate CV PDF (รัน script ในเครื่อง):

cd C:\Users\manito\Documents\portfolio-website
python scripts/generate_cv.py

แล้ว commit + push ใหม่ผ่าน GitHub Desktop

⚠ ข้อควรระวัง

  • Token เก็บใน browser ของท่าน — ถ้าใช้เครื่องสาธารณะให้กด "ออกจากระบบ"
  • การลบใน admin = commit ลบไฟล์บน GitHub → ย้อนกลับได้แค่ผ่าน git revert บน GitHub.com
  • แก้ไขผ่าน admin จะทำ commit แยกรายการ → ดู history ได้ที่ GitHub
